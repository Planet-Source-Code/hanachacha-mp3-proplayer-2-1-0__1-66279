* Upgarde from 2.0.0 :

  _ Allow using Winamp 2.x DSP plugin.
  _ Now use new format skin compress (*.skn).
  _ Run faster and just using 02 to 10 CPU proccess.
  _ Fixed all most bug found.
  _ Command line :
    /open 
    /play
    /next
    /previous
    /stop
    /add
    /exit
    /vold
    /volu
